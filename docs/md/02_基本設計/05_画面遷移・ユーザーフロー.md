## 2. 新規登録フロー

```
ランディングページ
    │ 「無料で始める」タップ
    v
┌────────────────────────────────────────────────────┐
│  新規登録（5ステップ）                                │
│                                                    │
│  Step 1: 基本情報                                    │
│  ├── 名前、ニックネーム                               │
│  ├── メールアドレス                                   │
│  ├── パスワード（8文字以上、英数字）                     │
│  └── パスワード確認                                   │
│      │                                              │
│  Step 2: 音楽プロフィール                              │
│  ├── 楽器（複数選択）★必須                             │
│  ├── ジャンル（複数選択）★必須                          │
│  └── スキルレベル                                     │
│      │                                              │
│  Step 3: コピバン情報                                 │
│  ├── コピーしたいアーティスト（Spotify検索）              │
│  ├── やりたい曲（Spotify検索）★必須                     │
│  └── 弾ける曲（Spotify検索）                           │
│      │                                              │
│  Step 4: 活動情報                                     │
│  ├── 都道府県（RESAS APIから選択）★必須                 │
│  ├── 市区町村（都道府県連動）                            │
│  └── 練習スケジュール（曜日・時間帯）★必須              │
│      │                                              │
│  Step 5: SNS連携 + 招待コード                          │
│  ├── SNS連携（X/Instagram/LINE/YouTube）※任意          │
│  └── 招待コード入力（トライアル延長）※任意               │
│      │                                              │
│  利用規約・コミュニティガイドライン同意                    │
│  □ 利用規約に同意します                                │
│  □ コミュニティガイドラインに同意します                    │
│  [同意して登録する]                                    │
│      │                                              │
│  クレジットカード登録                                   │
│  └── Stripe Checkout画面にリダイレクト                  │
│      （カード情報はStripeのサーバーで処理）               │
│      │                                              │
│  登録完了 → ダッシュボードへ                            │
└────────────────────────────────────────────────────┘
```

### ソーシャルログイン（代替フロー）

```
ランディングページ
    │ 「Googleで登録」/「Xで登録」/「Instagramで登録」
    v
Cognito Hosted UI（OAuth画面）
    │ 認証成功
    v
Step 2〜5（音楽プロフィール入力）
    │ ★ Step 1はソーシャルログインで自動入力
    v
利用規約同意 → Stripe Checkout → ダッシュボード
```

## 3. マッチングフロー

```
マッチング画面 (/matching)
    │
    ├── フィルター設定（地域、楽器、ジャンル等）
    │
    v
マッチング候補一覧（スコア順）
    │ ユーザーカードをタップ
    v
ユーザー詳細（プロフィール表示）
    │
    ├── [いいね] → いいね送信
    │       │
    │       ├── 相手もいいね済み → マッチング成立！
    │       │       │
    │       │       v
    │       │   マッチング成立画面
    │       │       │
    │       │       └── [チャットを始める]（Pro）→ チャット画面
    │       │
    │       └── 未マッチング → 候補一覧に戻る
    │
    └── [スキップ] → 次の候補へ
```

### 信頼レベルによる制限

```
Lv.0（登録直後）:
    マッチング検索は可能だが、いいね送信不可
    → 「プロフィールを完成させましょう」表示

Lv.1（プロフィール完成80%+）:
    いいね送信可能
    → Free: 3回/日、Pro: 無制限

Lv.2（マッチング成立1件+ & 登録3日+）:
    チャット送信可能（Pro）
```

## 4. コピバン結成フロー

```
バンド一覧 (/bands)
    │ [バンドを作成]
    v
バンド作成画面 (/bands/create)
    ├── バンド名
    ├── 説明
    ├── コピー対象アーティスト（Spotify検索）
    ├── ジャンル
    ├── 活動地域
    ├── 楽器スロット設定（ギター x2, ベース x1, ドラム x1 等）
    └── [作成]
        │
        v
バンド詳細画面 (/bands/[id])
    │
    ├── セットリスト管理
    │   ├── [曲を追加]（Spotify検索）
    │   └── 曲の順序変更・削除
    │
    ├── メンバー招待
    │   ├── マッチング済みユーザーから選択
    │   └── 招待通知が送信される
    │       │
    │       └── 招待を受けたユーザー
    │           ├── [参加する] → バンドメンバーに追加
    │           └── [辞退する] → 通知のみ
    │
    ├── 練習カレンダー（Pro）
    │   ├── [練習日程を提案]
    │   │   ├── 日時、スタジオ名、住所、メモ
    │   │   └── メンバーに通知
    │   ├── メンバーが OK/NG で回答
    │   └── リーダーが [確定] → カレンダーに表示
    │
    └── メンバー管理
        ├── メンバー一覧（役割・楽器表示）
        └── [メンバー除外]（リーダーのみ）
```

## 5. ライブ出演フロー

```
イベント一覧 (/events)
    │ フィルター: 地域、日付、初心者歓迎、ジャンル
    │ イベントをタップ
    v
イベント詳細 (/events/[id])
    ├── 会場、日時、料金、出演バンド一覧
    ├── 初心者歓迎バッジ
    ├── 「公式」バッジ（管理者登録イベント）
    │
    ├── [バンドで応募]（バンドリーダーのみ）
    │   ├── 自分のバンドを選択
    │   └── 応募完了 → イベント主催者に通知
    │
    └── [興味あり]（個人）
        └── 興味表明を記録

--- イベント作成（Pro + 管理者のみ） ---

イベント一覧
    │ [イベントを作成]（Proバッジ表示）
    v
イベント作成画面
    ├── タイトル、説明
    ├── 会場名、住所、都道府県
    ├── 日時
    ├── 料金
    ├── 初心者歓迎フラグ
    ├── ジャンルタグ
    └── [作成]
        │
        ├── 管理者が作成 → 「公式」バッジ付きで即公開
        └── Proユーザーが作成 → 管理者の承認後に公開（F-134）
```

## 6. 課金フロー

```
サブスクリプション画面 (/subscription)
    │
    ├── 現在のプラン表示（Free / Pro / トライアル中）
    ├── トライアル残日数
    │
    ├── [Proプランに変更]
    │   │
    │   ├── Web → Stripe Checkout にリダイレクト
    │   │       └── 決済完了 → success画面 → プラン更新
    │   │
    │   ├── iOS → App Store課金ダイアログ
    │   │       └── 購入完了 → レシート検証 → プラン更新
    │   │
    │   └── Android → Google Play課金ダイアログ
    │           └── 購入完了 → トークン検証 → プラン更新
    │
    └── [プランを解約]
        ├── 確認ダイアログ「本当に解約しますか？」
        ├── 解約理由の選択（任意）
        └── 解約完了 → 次回更新日までPro継続 → 更新日にFreeへ
```

## 7. 管理画面フロー

```
管理画面 (/admin) ※管理者権限必須
    │
    ├── ダッシュボード
    │   ├── 総ユーザー数、アクティブ数、Pro会員数
    │   ├── 月間売上グラフ
    │   └── 通報件数（未対応）
    │
    ├── ユーザー管理 (/admin/users)
    │   ├── ユーザー検索・一覧
    │   ├── プラン変更
    │   ├── アカウント停止/BAN
    │   └── 通報履歴確認
    │
    ├── イベント管理 (/admin/events)
    │   ├── Proユーザー投稿の承認・却下
    │   └── イベント削除
    │
    ├── 通報管理
    │   ├── 通報キュー（未対応一覧）
    │   ├── 通報詳細（カテゴリ、対象、メッセージ履歴）
    │   └── 対応アクション（警告/制限/停止/BAN）
    │
    └── 売上管理 (/admin/revenue)
        ├── 売上推移グラフ
        └── サブスクリプション内訳（Free/Pro/トライアル）
```

## 8. モバイルナビゲーション

### タブバー（画面下部・5タブ）

```
┌──────┬──────┬──────┬──────┬──────┐
│ ホーム │マッチ│バンド │コミュ│プロフ│
│  🏠  │  💕  │  🎸  │  📝  │  👤  │
└──────┴──────┴──────┴──────┴──────┘
  /dashboard /matching /bands /community /profile
```

| タブ | 遷移先 | 説明 |
|------|--------|------|
| ホーム | /dashboard | ダッシュボード、お知らせ |
| マッチ | /matching | マッチング候補一覧 |
| バンド | /bands | バンド一覧 + イベント一覧 |
| コミュ | /community | タイムライン |
| プロフ | /profile | 自分のプロフィール、設定 |

### ヘッダー（画面上部）

```
┌────────────────────────────────┐
│ [←] Groove Match    🔔(3)  ⚙️ │
└────────────────────────────────┘
  戻る   ロゴ     通知   設定
```
