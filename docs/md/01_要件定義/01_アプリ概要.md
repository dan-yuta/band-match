# Groove Match - アプリ概要

## Groove Match とは？

**Groove Match** は、好きなバンドの曲をコピーしたい初心者が、同じ曲をやりたい仲間を見つけてコピバンを結成し、コピバンライブに出るまでをサポートするアプリです。

**クロスプラットフォーム（Web / iOS / Android）対応コミュニティアプリ**として提供します。

> **本アプリのゴールは「コピバンを組んでライブに出ること」です。**
> メインフローは **仲間探し → コピバン結成 → ライブ出演** の一気通貫体験であり、練習トラッカーやフレンド機能はそのゴールに到達するまでの挫折を防ぐサポート機能として位置づけています。

**やりたい曲・アーティストベースのコピバンメンバーマッチング**と**ライブ出演の促進**を軸に、月額課金で運営します。

## ビジョン：Groove Match が解決する5つの課題

コピバンをやりたい初心者が直面する問題と、Groove Matchの解決策を示します。

### メインフロー（仲間探し → コピバン結成 → ライブ出演）

| 課題 | 解決策 |
|------|--------|
| 同じ曲をやりたい仲間がいない | やりたい曲・アーティストで**曲ベースマッチング**（Spotify API連携で曲を検索） |
| コピバンの組み方が分からない | **コピバン作成・セットリスト管理**で簡単にバンド結成 |
| レベルが合わない | スキルレベルを考慮した**スコアリング**で適切な相手を提案 |
| ライブに出られない | **初心者歓迎のコピバンイベント**を掲載。運営+Proユーザーが登録可能（運営イベントには「公式」バッジ付き） |

### 挫折防止サポート機能

| 課題 | 解決策 |
|------|--------|
| 挫折する | **Duolingo型練習トラッカー**（ストリーク・バッジ・ランキング）で継続を支援 |
| 仲間との繋がりが薄い | **相互承認型フレンド機能**で練習状況を共有。フレンド間の週間ランキングで切磋琢磨 |
| 活動をアピールしにくい | **SNS連携**（X / Instagram / LINE / YouTube）でバンド活動を発信 |
| アカウント作成が面倒 | **ソーシャルログイン**（Google / X / Instagram）でワンクリック登録 |

## ペルソナ

### ペルソナ1: 大学生ギタリスト（田中 太郎・20歳）
- 大学入学を機にギターを始めた
- BUMP OF CHICKENの「天体観測」を弾けるようになりたい
- 軽音サークルには入ったが、やりたい曲が合うメンバーが見つからない
- **Groove Matchで解決**: Spotify APIで「天体観測」を検索して登録 → 同じ地域のベース・ドラムとマッチング → コピバン結成

### ペルソナ2: 社会人ドラマー（鈴木 花子・28歳）
- 高校時代に軽音部でドラムを担当、社会人になって再開
- ONE OK ROCKやRADWIMPSのコピーをやりたい
- 社会人バンドサークルを探しているが、日程が合わない
- **Groove Matchで解決**: スケジュールマッチングで週末練習可能なメンバーを発見 → ワンオクコピバン結成

### ペルソナ3: 初心者ベーシスト（山田 健太・24歳）
- ベースを始めて3ヶ月、まだ自信がない
- アニソン（けいおん!、ぼっち・ざ・ろっく!）のコピーがしたい
- 経験者ばかりの場所だと萎縮してしまう
- **Groove Matchで解決**: 同じ初心者レベル＋アニソン好きでマッチング → 初心者同士のコピバンで安心してライブを目指す

## プロフィールの年代表示

ユーザーのプロフィールでは、生年月日から計算して**「20代前半」「30代後半」**のように年代で表示します。生年月日そのものは非公開です。これにより、プライバシーを守りつつ、年齢感の近い仲間を見つけやすくしています。

## ビジョン・ミッション

> **「やりたい曲で仲間を見つけて、コピバン組んで、ライブに出よう」**

| 区分 | 内容 |
|------|------|
| ビジョン | 「好きな音楽で繋がる」コミュニティを全国に広げ、アマチュアバンドシーンを活性化する |
| ミッション | バンドメンバー探しの非効率（SNS検索、掲示板投稿の手間）をなくし、マッチングから本番ライブまでを一気通貫でサポートする |
| ポジショニング | コピーバンド特化型（既存の汎用ミュージシャンマッチングとの差別化） |
