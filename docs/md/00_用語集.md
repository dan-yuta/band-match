# Groove Match - 用語集

プロジェクト全体で使用する技術用語・ビジネス用語の一覧です。

---

## ビジネス用語

| 用語 | 読み | 説明 |
|------|------|------|
| **MAU** | エムエーユー | Monthly Active Users（月間アクティブユーザー数）。1ヶ月に1回以上アプリを利用したユーザーの数 |
| **DAU** | ディーエーユー | Daily Active Users（日間アクティブユーザー数）。1日に1回以上アプリを利用したユーザーの数 |
| **DAU/MAU比** | ─ | エンゲージメント指標。値が高いほど毎日利用するユーザーが多いことを示す（目標: 30%以上） |
| **KPI** | ケーピーアイ | Key Performance Indicator（重要業績評価指標）。目標達成度を測る定量的な指標 |
| **MVP** | エムブイピー | Minimum Viable Product（最小限の機能を持つ製品）。最小構成でリリースし、フィードバックを得て改善する開発手法 |
| **NPS** | エヌピーエス | Net Promoter Score（推奨度）。「このサービスを友人に勧めますか？」の回答から算出する顧客満足度指標 |
| **Pro会員** | プロかいいん | 本アプリの有料プランユーザー。月額 ¥500 で無制限マッチング・チャット等の拡張機能を利用可能 |
| **コピーバンド** | ─ | 既存アーティストの楽曲を演奏するバンド。略称「コピバン」。本サービスの主要ターゲット |
| **レピュテーションスコア** | ─ | ユーザーの信頼度を示す内部スコア（0〜100）。通報・違反で低下し、問題なし利用で自然回復 |
| **SLA** | エスエルエー | Service Level Agreement（サービスレベル合意）。対応時間・可用性等のサービス品質を定めた合意事項 |

## 技術用語

| 用語 | 読み | 説明 |
|------|------|------|
| **API** | エーピーアイ | Application Programming Interface。ソフトウェア同士が通信するためのインターフェース規格 |
| **REST API** | レストエーピーアイ | HTTPを使ったAPIの設計スタイル。GETやPOSTなどのHTTPメソッドを用いる |
| **JWT** | ジェーダブリューティー | JSON Web Token。認証情報をJSON形式でエンコードしたトークン。Cognitoが発行する |
| **ORM** | オーアールエム | Object-Relational Mapping。コードでDBを操作するための仕組み（本プロジェクトでは Prisma を使用） |
| **CDN** | シーディーエヌ | Content Delivery Network。世界各地のサーバーにコンテンツを配置し、ユーザーに近いサーバーから高速配信する仕組み（AWS CloudFront） |
| **WebSocket** | ウェブソケット | サーバーとクライアント間でリアルタイム双方向通信を行うプロトコル。チャット機能に使用 |
| **Embedding** | エンベディング | テキストをベクトル（数値列）に変換する技術。ローンチからハイブリッドマッチングに使用 |
| **pgvector** | ピージーベクター | PostgreSQLにベクトル検索機能を追加する拡張機能。Embedding類似度計算に使用 |
| **CI/CD** | シーアイシーディー | Continuous Integration / Continuous Delivery。コード変更を自動テスト・デプロイする仕組み |
| **IaC** | アイエーシー | Infrastructure as Code。インフラ構成をコードで定義・管理する手法 |
| **PCI DSS** | ピーシーアイディーエスエス | Payment Card Industry Data Security Standard。クレジットカード業界のセキュリティ基準 |
| **RTO** | アールティーオー | Recovery Time Objective（目標復旧時間）。障害発生からサービス復旧までの目標時間 |
| **RPO** | アールピーオー | Recovery Point Objective（目標復旧時点）。障害発生時に許容できる最大データ損失時間 |

## AWSサービス

| 用語 | 読み | 説明 |
|------|------|------|
| **ECS Fargate** | イーシーエス ファーゲート | サーバーレスコンテナ実行環境。サーバー管理不要でDockerコンテナを動かせる |
| **RDS** | アールディーエス | Amazon Relational Database Service。マネージドRDB（本プロジェクトでは PostgreSQL を使用） |
| **Aurora** | オーロラ | AWS独自のマネージドRDB。RDS互換でより高いパフォーマンス・可用性を持つ（MAU 5,000超で移行予定） |
| **ElastiCache** | エラスティキャッシュ | マネージドキャッシュサービス（Redis互換）。マッチングスコアのキャッシュ等に使用 |
| **Cognito** | コグニト | AWS認証サービス。ユーザー登録・ログイン・ソーシャルログイン連携を提供 |
| **CloudFormation** | クラウドフォーメーション | AWSのIaCサービス。YAMLまたはJSONでAWSリソースを定義・デプロイできる |

## フレームワーク・ライブラリ

| 用語 | 読み | 説明 |
|------|------|------|
| **Prisma** | プリズマ | Node.js / TypeScript向けORM。型安全にデータベースを操作できる |
| **Capacitor** | キャパシター | WebアプリをiOS/Androidのネイティブアプリとしてラップするフレームワーク |
| **Stripe** | ストライプ | オンライン決済サービス。クレジットカード決済・サブスクリプション管理に使用 |
