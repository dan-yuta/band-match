マッチングのAI利用方針

Version A（AI不使用）とVersion B（AI利用）の2バージョンを用意

Version A: ルールベース（AI不使用）

方式: 6要素の加重スコアリングのみ
追加コスト: ¥0
対象: Freeプラン
利点: コストゼロ・決定論的・説明可能

Version B: ハイブリッド（AI利用）

方式: ルールベース×0.85 + Embedding類似度×0.15
AIモデル: OpenAI text-embedding-3-small（推奨）or AWS Bedrock Titan V2
追加コスト: ¥3,003〜5,060/月
対象: Proプラン限定
精度向上: +8〜15%
Phase 2（MAU 2K〜5K）: LLM推薦テキスト追加。「なぜマッチしたか」をPro会員に自然言語で説明
