🔍 基本SEO設定

✅title タグ — ページごとに固有
✅meta description — 120文字以内
✅OGP — og:title, og:description, og:image
✅Twitter Card — twitter:card, title, image
✅canonical URL — 重複コンテンツ防止
✅sitemap.xml — 公開ページ自動生成
✅robots.txt — 管理画面・APIを除外
✅構造化データ — JSON-LD（Eventスキーマ）

⚡ 技術的SEO

✅SSR — ランディング・イベント一覧はSSRで配信
✅Core Web Vitals — LCP < 2.5s, FID < 100ms, CLS < 0.1
✅モバイルフレンドリー — レスポンシブデザイン
✅HTTPS — 全ページHTTPS
✅CDN — CloudFront + next/image最適化
